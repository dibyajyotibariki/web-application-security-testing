# Web Application Security Testing - Week 5 & 6

This project covers basic web application security testing using ethical hacking methods.

## Topics Covered
- DVWA / OWASP Juice Shop setup
- OWASP Top 10 vulnerabilities
- SQL Injection
- Cross-Site Scripting (XSS)
- Broken Authentication
- Security Misconfiguration

## Tools Used
- DVWA or OWASP Juice Shop
- XAMPP / Docker / Node.js
- Browser Developer Tools
- Burp Suite Community Edition

## Disclaimer
This project is for educational and ethical learning purposes only.
